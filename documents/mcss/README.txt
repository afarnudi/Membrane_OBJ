Run "doxygen doxy_conf" to create html files.
----------------------------------------
In doxy_conf:
HTML_EXTRA_STYLESHEET  = "custom.css" \
			 "custom_dark_theme.css"
HTML_EXTRA_FILES       = "doxy-boot.js"
HTML_HEADER            = "html_header.html"
HTML_FOOTER            = "html_footer.html"
----------------------------------------
Edit custom_dark_theme.css and custom.css to change color and theme.
