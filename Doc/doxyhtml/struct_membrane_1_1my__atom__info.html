Detailed Description

This is a struct we can use to collect atom parameters from the config files.

We're going to use data already imported into the class and convert to OpenMM's internal unit system:

pdb mass charge vdwRad vdwEnergy gbsaRad gbsaScale initPos example: {" NA ", 22.99, 1, 1.8680, 0.00277, 1.992, 0.8, 8, 0, 0}
